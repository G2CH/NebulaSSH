## ğŸš€ Getting Started

### Prerequisites

- Node.js (v18+)
- Rust (latest stable)
- Tauri CLI

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/nebula-ssh.git
   cd nebula-ssh
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run in development mode**
   ```bash
   npm run tauri dev
   ```

4. **Build for production**
   ```bash
   npm run tauri build
   ```
